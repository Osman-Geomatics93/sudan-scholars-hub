## Suggested Content for Each Guide

### Bachelor's Guide
- Eligibility requirements (age under 21, 70% GPA minimum)
- Required documents list
- Step-by-step application process
- Timeline and deadlines
- Tips for personal statement

### Master's Guide
- Eligibility requirements (age under 30, 75% GPA minimum)
- Work experience requirements
- Research proposal tips
- Interview preparation
- Recommendation letter guidance

### PhD Guide
- Eligibility requirements (age under 35, 75% GPA minimum)
- Research proposal writing guide
- Supervisor matching tips
- Academic CV preparation
- Publication requirements

### Document Checklist
- Passport/ID requirements
- Academic transcripts
- Diploma/certificates
- Language proficiency proof
- Photos specifications
- Health certificate
- Recommendation letters

### Application Tips
- How to write a compelling personal statement
- Common mistakes to avoid
- Interview tips
- Scholarship essay examples
- Success stories

### Recommendation Letters
- Letter format and structure
- How to approach recommenders
- What information to include
- Sample phrases and expressions
- Specific requirements for each level (BSc/MSc/PhD)

### Validation Letters
- Letter format for enrollment verification
- Required elements (student info, enrollment dates, program details)
- Official stamp/signature guidelines
- Translation requirements
- Sample templates

## File Format Requirements

- Format: PDF
- Max file size: 10MB per file
- Language: Use clear, simple language
- Accessibility: Include text (not just images) for screen readers
